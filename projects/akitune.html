Akitune — Adaptive Audio Retuning System

Python • Deep Learning • Audio Processing

Features & Functions

  • Overstimulating Audio Detection: Akitune scans audio content from videos and media to identify sections with excessively high volume or harsh frequencies. This helps protect children and sensitive listeners from potential auditory overstimulation.
  • Advanced Audio Feature Extraction: The system uses STFT and MFCC to analyze audio signals in detail. These techniques allow accurate identification of pitch, volume, and frequency patterns for precise detection.
  • Deep Learning Classification: A VGG16 CNN model classifies audio segments based on intensity and overstimulation risk. It is trained on diverse media to reliably detect loud or harsh content.
  • Automatic Audio Retuning: Detected overstimulating audio is adjusted automatically. Peaks are softened, and harsh frequencies are balanced to create a safer listening experience while preserving clarity.
  • Personalized User Profiles: Users can set age-specific hearing sensitivity levels. The system applies retuning rules based on these profiles and maintains logs of detected audio events.
  • Real-Time Feedback and Logging: Akitune generates logs showing overstimulating segments and applied adjustments. Parents, educators, and users can review these logs to monitor audio safety.
  • Compatibility Across Media Types: The system supports various audio formats, including streamed and local files. This ensures flexible use across different platforms and media types.
